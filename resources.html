    • Resources to support exploration into play data:

    • ADAGE API

      An open-source data collection and analysis framework to transform telemetry data into evidence of learning.

    • D3

      A JavaScript library for manipulating documents based on data. D3 helps you bring data to life using HTML, SVG and CSS.

    • IPYTHON

      Architecture designed in a language-agnostic way to facilitate interactive computing in any language.

    • GGPLOT

      A plotting system for Python based on R's ggplot2 and the Grammar of Graphics. It is built for making profressional looking, plots quickly with minimal code.

    • PANDAS

      An open source, BSD-licensed library providing high-performance, easy-to-use data structures and data analysis tools for the Python programming language.

    • Analyzing Data with Adage

      Book aimed at anyone interested in studying data from learning games, whether they be graduate students, professional researchers, game designers, software engineers, or really anyone passionate about games. Written by Brian A. Danielak.
